# Writing Prompts

## Draft a Professional Email

**Use case:** Compose a clear, polite email for a workplace situation.

**Prompt:**
```
Write a professional email with the following details:

- From: <your name / role>
- To: <recipient name / role>
- Subject: <topic>
- Purpose: <what you want to accomplish, e.g., request a meeting, follow up on a deliverable>
- Key points to include:
  - <point 1>
  - <point 2>
- Tone: <formal / semi-formal / friendly>

Keep it concise (under 150 words) and end with a clear call to action.
```

---

## Write an Executive Summary

**Use case:** Condense a project or report into a short summary for leadership.

**Prompt:**
```
Write a 200-word executive summary for the following project:

Project name: <name>
Goal: <what the project aims to achieve>
Key results / findings:
- <result 1>
- <result 2>
- <result 3>
Recommendations: <what action should be taken next>

The audience is senior leadership with limited technical background. Use plain language.
```

---

## Create a Meeting Agenda

**Use case:** Produce a structured agenda for an upcoming meeting.

**Prompt:**
```
Create a meeting agenda for a <duration>-minute <meeting type, e.g., weekly team sync / project kickoff> meeting.

Attendees: <list of roles>
Goal of the meeting: <what should be decided or accomplished>
Topics to cover:
1. <topic 1>
2. <topic 2>
3. <topic 3>

Format the agenda with time allocations and indicate who leads each section.
```

---

## Write a Status Update

**Use case:** Summarize progress on a project for stakeholders.

**Prompt:**
```
Write a brief weekly status update for the following project:

Project: <project name>
Reporting period: <date range>
Completed this week:
- <item 1>
- <item 2>
Planned for next week:
- <item 1>
- <item 2>
Blockers / risks: <any issues or dependencies>

Keep it under 150 words and use a bullet-point format.
```
