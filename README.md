# Prompt Engineer

A collection of example prompts for common work tasks. Use these as starting points and adapt them to your specific needs.

## Categories

- [Coding](prompts/coding.md) – Code generation, review, debugging, and documentation
- [Writing](prompts/writing.md) – Drafting emails, reports, and other written content
- [Data Analysis](prompts/data_analysis.md) – Summarizing data, creating queries, and interpreting results
- [Summarization](prompts/summarization.md) – Condensing long documents and extracting key points

## Usage

Open any category file, copy the prompt template that matches your task, fill in the placeholders (marked with `<angle brackets>`), and paste it into your preferred LLM (e.g., ChatGPT, Claude, Copilot).

## Contributing

Feel free to add new prompts by opening a pull request. Follow the existing format:

```
## <Title>

**Use case:** <one-line description>

**Prompt:**
\`\`\`
<prompt text with <placeholders> for variable parts>
\`\`\`
```
